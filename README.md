# Bike Sales Analysis & Dashboard

**Tools Used:** Excel (Pivot Tables, Formulas, Dashboards, Slicers)  
**Context:** Portfolio Project  

## Project Summary

Analyzed a bike sales dataset to understand how factors like income, gender, age, and commute distance influence purchasing behavior. Cleaned and standardized the data, created age brackets, and built pivot tables to explore key trends. Developed an interactive Excel dashboard with slicers to visualize findings, helping stakeholders identify potential customer segments and make data-driven sales and marketing decisions.

## Key Features & Workflow

### Data Cleaning & Transformation
- Standardized `Marital Status` (M → Married, S → Single) and `Gender` (F → Female, M → Male) for clearer stakeholder readability  
- Created an `Age Brackets` table using nested IF statements to categorize customers into age groups for more concise visualizations  

### Pivot Tables & Visualizations
- **Pivot Table 1:** Bike purchases by average income and gender 
- **Pivot Table 2:** Impact of commute distance on bike sales  
- **Pivot Table 3:** Age group distribution for customers who purchased vs. didn’t purchase bikes  

### Dashboard Creation
- Combined all key visualizations into a single interactive dashboard 
- Added slicers to filter by income, gender, age group, and commute distance for enhanced stakeholder interactivity  

## Skills Demonstrated
- Data cleaning and transformation using Excel formulas  
- Pivot table creation and multi-dimensional analysis  
- Dashboard design and interactive reporting using slicers  
- Translating data insights into actionable business recommendations  

## Outcome & Impact
- Identified high-value customer segments by income, gender, and age group  
- Provided stakeholders with interactive, easy-to-read dashboards for quick insights  
- Enabled targeted marketing strategies and improved sales forecasting
